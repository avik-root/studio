# **App Name**: ColorVisionary

## Core Features:

- Home Screen: Home screen with 'La interior' branding and a prominent 'Sign In' button.
- User Authentication: User authentication via username, password, and a 6-digit PIN [using mock IDs for MVP].
- Capture/History Options: Two options upon login: 'Use Camera/Gallery' to capture a wall photo, and 'History' to view past projects.
- Color Selection Mode: Options after capturing a photo: 'Use AI Suggestion' for color palettes, or 'Choose Your Own' to manually select colors.
- AI Color Replacement: AI-powered analysis of the captured photo and application of the selected/suggested colors using colors fetched from Berger Paints color codes.
- Sheen Suggestion: AI powered "tool" provides suggestions on preferred paint sheens (e.g., silk, matte, etc.) based on the selected color and wall photo analysis.
- Color Matching Suggestion: AI "tool" provides suggestions for complementary colors based on the user's chosen color and the captured room photo, creating a matched color palette.
- History Logging: Save colorized outputs, user color choices and room photos in a project history.
- Theme Support: Support for both dark mode and light mode themes.
- AI Dashboard: Dashboard for easy access and use of AI features

## Style Guidelines:

- Primary color: A soft, muted blue (#A0BBD9) to represent trust and innovation in interior design.
- Background color: Very light desaturated beige (#F5F5DC), creating a calming backdrop.
- Accent color: A gentle lavender (#E6E6FA) to highlight AI-driven features and add a creative touch.
- Clean and modern typography for a user-friendly interface.
- Intuitive icons to represent app functions and features.
- Simple and organized layout to ensure ease of navigation.
- Smooth transitions and animations to enhance the user experience.
- Dark mode palette: Dark blues and grays for reduced eye strain in low-light environments.
